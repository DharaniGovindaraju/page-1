Pipes

Pipes in Angular 2 will get data as input and format data required way and render in your templates. Some of predefined pipes also available dates, currency, percentage and character cases etc. you can also define custom pipes of your own easily.

Prerequisites

Angular Cli must be installed if not installed check out my previous article
“Install and run Angular 2 apps using Angular CLI”.

Some basic knowledge on Html, Css, Javascript, Oops.

Create a new pipe

Using command :

ng generate pipe pipe-name or ng g p pipe-name

Here we are going to create pipe to reverse character in a word.

ng generate pipe reversechar

it will automatically generate default ts file with code related and configure reversechar pipe in app.module file.

Manually :

Inside app folder create a new typescript file with name as given below

reversechar.pipe.ts

app.module.ts

Add the custom pipe as a declaration in your app module:

app.component.html

And finally use custom pipe in your view

Analysing reversechar.pipe.ts

import { Pipe, PipeTransform } from ‘@angular/core’;

Imported the Pipe, PipeTransform decorator from Angular core module.

@Pipe({name: ‘reverseChar’})

@Pipe() decorator will tell the angular that this file used to define custom pipe.

{name: ‘reverseChar’} parameter inside @Pipe() is the pipe name we are going to

use in our template(

{{ username | reverseChar }}
).

export class ReverseChar implements PipeTransform {

transform(value: string): string {

let newString: string = “”;

for (var i = value.length — 1; i >= 0; i — ) {

newStr += value.charAt(i);

}
}

return newString;

}
}

Final step : exported class which contains transform method that accepts a data as input and returns transformed or formatted value as output.
